# Web Player

This is a real production web player extracted from a [live project](https://player.radio10.fi). It is **not** a general-purpose, drop-in solution — it will require significant modification to work in a different environment.

If you're looking for a starting point to build a simple web player, this codebase can serve as a reference or foundation to build on top of.
